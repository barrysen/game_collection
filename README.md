## 下载地址

- [最新版本下载](https://github.com/barrysen/game_collection/releases/latest)
- [历史版本](https://github.com/barrysen/game_collection/releases)

## 游戏说明

### 贪吃蛇
- 使用方向键控制蛇的移动
- 空格键暂停游戏
- 吃到食物可以增加长度和分数

### 俄罗斯方块
- 方向键左右移动方块
- 上方向键旋转方块
- 下方向键加速下落
- 空格键直接落下

## 安装说明

### Windows用户
1. 下载 `经典游戏合集.exe`
2. 双击运行即可

### Mac用户
1. 下载 `经典游戏合集.app`
2. 双击运行即可

## 注意事项
- 游戏会自动保存最高分记录
- 如遇到字体显示问题，请确保系统安装了中文字体
